Alex's Styling

This text is in a p tag, so it can go over multiple lines easily. The background of this text is steel blue, Jack Polifka's favorite color. I used the background-color property to change it. I changed the padding to 7 pixels so the background was not right up against the text. The font-style is oblique and the letter-spacing has been set to 1.5 pixels so it is easier to read. The color property is white so the text is easier to see on the steel blue background.
